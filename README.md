Bay-area-housing-price-visualization-
=====================================

A visualization of Housing Price in Bay Area
-------------------------

### Details
    1. Analysis and process the data by R Language
    2. Design suitable visualize  methods
    3. format the unorganized data and generate the objective JSON file by Python
    4. Visualize those JSON files by D3.js, Html, CSS and Jquery
